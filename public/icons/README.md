# PWA Icons Placeholder

Для полноценной PWA нужны PNG-иконки следующих размеров:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

Генерация из SVG:

```bash
npx pwa-asset-generator public/icons/icon.svg public/icons
```

Или используйте онлайн-генераторы:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
```

Временно используется SVG-заглушка.
